웹 서버에 대해서

웹호스팅에 업로드하지 않고 노트북에서 작업한 문서를 동일한 ip서버를 통해 모바일에서 확인하는 방법

  1. VS CODE 플러그인에서 IP Adress를 인스톨하고 하단에 우리집 ip주소가 떴는지 확인

  2. 그런 다음에 모바일에서 ip 주소를 입력하고 뒤에 ' :5500 '을 추가하면 폰에서도 해당 웹을 볼 수 있다.

  3. 단, 노트북과 핸드폰이 동일한 와이파이여야 할 것!
